2. Operační systémy, základní funkce, možnosti konfigurace uživatelského rozhraní

Co je to operační systém?

  • SW - zpřístupnit a spravovat výpočetní zdroje HW
  • Výpočetní zdroje:
    • CPU
    • RAM
    • Souborový systém
    • I/O zařízení
  • Uživatelské rozhraní (UI)
  • Vytvoří prostředí pro uživatele

UI - User Interface

CLI - Command Line Interface

  • Vše na obrazovce je zobrazováno v text
  • Dodnes je součástí OS, ale ne jako hlavní rozhraní
  • Např. MS-DOS, Cisco

TUI - Text User Interface

  • Lepší než příkazová řádka (CLI), ale horší než GUI
  • Vše na obrazovce je zobrazováno v řádcích a sloupcích
  • Pomocí speciálních znaků se může napodobit GUI (okna, menu, ...)

GUI - Graphic User Interface

  • Okna, tlačítka, myš, ...
  • Např. Windows 7, 8, 10, ...

API - Application programming interface

  • Rozhraní pro programování aplikací, pro odesílání a přijímání dat
  • Typy: REST, SOAP, WebSocket, POSIX

GNOME

  • GUI pro Linux
  • Starší styl, blíže k Linuxu

KDE

  • GUI pro Linux
  • Moderní styl, blíže k Windows nebo MacOS

Kernel

  • Typy:
    • Mikro jádro
    • Monolitické jádro - více funkcí
  • Základní společné funkce:
    • Přerušení
    • Časovače
    • IPC
      • Typy:
        • Pipe - proud dat
        • Message - všechny data najednou
        • Socket
  • Kernel
    • Režimy mapování CPU (řazení/omezení oprávnění)
      • Ring
        • Ring0 - kernel
        • Ring1
        • Ring2
        • Ring3 - user
  • User - systémové procesy, běžná aplikace
  • Monolitické jádro
    • Aplikace - volání - knihovna - register - systémové volání - jádro OS - vrací:
      • -1 - chyba
      • 0 - standartní hodnota
  • HW - ovladače
  • RAM - správa paměti
  • Soubory - specializace
  • Spravovat procesy

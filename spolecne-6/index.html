6. Aktivní a pasivní síťové prvky, typy, parametry, konfigurace

Aktivní a pasivní prvky

Pasivní

  • Hub, kabel

Aktivní

  • Fyzická aktivita
  • Logické služby

Prvky v modelu

  • Aplikační - PC, Firewall
  • Síťová vrstva - Router
  • Síťová rozhraní - Switch, Hub, Bridge
  • Fyzická - Reapeter

Rozdíl mezi Switch a Hub

  • Hub kopíruje všude
  • Switch má logiku

Broadcast

  • Oslovuji všechny v síti
  • FF:FF:FF:FF:FF:FF
  • ARP
    • Spojuje MAC s IP adresou
  • Dělí je router

Kolize

  • Srazí se nám signály
  • Switch zabraňuje kolizi
  • Při použití full duplex nevznikají kolize
  • Řešení:
    • CSMA/CD - detekuje chyby
    • CSMA/CA - předchází chybám

Gateway

  • Dělí broadcast
  • Bridge pro LAN sítě
  • Komunikační brána
  • Standartní služby - DHCP, DNS, VPN, SSH

Firewall

  • Způsob ochrany sítě
  • Whitelist - Povolené
  • Blacklist - Zakázané
  • Filtruje:
    • Aplikační - protokoly
    • Transportní - SSH -> port 22
    • Síťová - IP -> ACL - přístupové seznamy (IP -> IP př)

Porty

  • Kategorie:
    1. Známé porty - 0 až 1023 - vyhrazené pro nejběžnější služby
    2. Registrované porty - 1024 až 49151 - použití portu by se mělo registrovat
    3. Dynamické porty - 49152 až 65535 - nejsou pevně přiděleny
  • Známé porty:
    • 22 - SSH
    • 23 - Telnet
    • 53 - DNS
    • 80 - HTTP
    • 443 - HTTPS

DHCP

  • Typy:
    • IPv4:
      1. DHCP
    • IPv6:
      1. DHCPv6
      2. SLAAC - server pošle default gateway a prefix, počítač si dopočítá IP adresu
      3. SLAAC s DHCPv6 - od SLAAC dostane default gateway a prefix, od DHCPv6 dostane IP adresu
